# 📊 Dashboard de Análise de Salários na Área de Dados

## 📌 Sobre o Projeto
Este projeto consiste em um dashboard interativo desenvolvido em Python com o objetivo de analisar salários na área de dados ao longo dos últimos anos.

A aplicação permite ao usuário explorar os dados de forma dinâmica, utilizando filtros e visualizações gráficas para identificar padrões e tendências.

---

## 🚀 Funcionalidades

- Filtros interativos por:
  - Ano
  - Senioridade
  - Tipo de contrato
  - Tamanho da empresa

- Métricas principais:
  - Salário médio
  - Salário máximo
  - Salário mínimo
  - Total de registros
  - Cargo mais frequente

- Visualizações:
  - Top 10 cargos por salário médio
  - Distribuição de salários (histograma)
  - Proporção dos tipos de trabalho (pizza)
  - Salário médio por país (mapa)

- Tabela com dados detalhados

---

- Plotly
